GenSlide One-Click Production Build

Steps:
1. Replace public/index.html with your GenSlide UI file
2. Add .env file:
   GEMINI_API_KEY=your_api_key_here
3. Push to GitHub
4. Connect to Netlify
5. Add GEMINI_API_KEY in Netlify environment variables
6. Deploy

This is full SaaS serverless architecture.
